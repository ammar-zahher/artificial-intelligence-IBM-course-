# IBM AI Engineering Course Projects 🚀

In my IBM AI Engineering Professional Certificate course, I use Python and machine learning techniques to solve practical issues.

## 📱 Predict Customer Churn (Logistic Regression)
The task here was simple: to predict customer churn within a telecommunication company.

My process:
- Data preparation: After analyzing coefficients, I discarded attributes such as address and income.
- Hyperparameter tuning: I experimented with different values, settling on C = 100 using liblinear.
- Output: The model achieved 80% accuracy with a log loss of 0.46.

## 📈 Regression Models
- Multiple Linear Regression: Designed to incorporate multiple predictors in its calculations.
- Simple Linear Regression: Created to establish relationships between one predictor and the dependent variable.

## 🛠 Tools Used
- Programming: Python (Pandas, NumPy)
- Machine Learning: Scikit-Learn (LogisticRegression, StandardScaler, train_test_split)
- Visualization: Matplotlib
